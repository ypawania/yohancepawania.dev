# Personal Site Design Guide

## Goal
Build a minimal, text-first personal website with a refined, human feel. The site should emphasize achievements, clarity, and taste rather than flashy visuals.

## Overall style
- Dark background
- No gradients
- No purple
- Minimal color usage
- No cards unless absolutely necessary
- No glassmorphism
- No large illustrations
- No heavy animations
- Text-first, understated, editorial feel
- Should feel like a thoughtful personal site, not a startup landing page

## Layout
- Single centered column
- No scroll on desktop for the landing page
- Top horizontal navbar
- Section for short intro / about text
- Section for bullet-point achievements
- Textual social links near the bottom
- Responsive on mobile by tightening spacing and font sizes, not by changing the whole design

## Typography
- Clean sans-serif font
- Strong hierarchy through size and weight, not color
- Generous spacing
- Comfortable paragraph line length
- Keep headings simple and restrained
- Avoid oversized hero text

## Color system
- Background: near black
- Primary text: off-white
- Secondary text: muted gray
- Links: white or slightly brighter gray
- Borders: subtle gray only if needed

## Content tone
- Intelligent, direct, understated
- Confident but not try-hard
- Human and personal, not corporate
- Focus on building, technical depth, and achievements
- Avoid buzzwords and generic startup language

## Do not add
- gradients
- purple accents
- cards everywhere
- glowing effects
- unnecessary icons
- stock illustrations
- fake testimonials
- overengineered sections
- flashy animations
- typewriter effects
- metric counters

## Tailwind style direction
- Use neutral color palette
- Prefer spacing and typography over decoration
- Use max-width centered layout
- Keep sections vertically spaced
- Keep hover states subtle

## Spacing system

Use a consistent spacing scale:
- gap-2: tight spacing
- gap-4: normal spacing
- gap-8: section spacing
- gap-12: major separation

Use vertical rhythm:
- space-y-10 or space-y-12 for main layout

## Layout

- Centered column layout
- max-w-2xl or max-w-3xl
- horizontal padding px-6

## Typography

- Use a clean sans-serif font (Inter or Geist)
- Name: text-2xl to text-3xl, font-semibold
- Body: text-sm to text-base
- Secondary text: text-neutral-400
- Use leading-relaxed for readability

## Colors

- Background: neutral-950
- Primary text: neutral-100
- Secondary text: neutral-400
- Avoid bright or saturated colors

## Components

Navbar:
- text-sm
- subtle, low contrast
- no backgrounds or borders

Bullet lists:
- compact
- high signal
- space-y-2 or space-y-3
